# 화면 설계 (Wireframes)

## 1. 메인 캘린더 뷰

```
┌─────────────────────────────────────────────────────┐
│  카나다광림교회 공간 신청 시스템                          │
├─────────────────────────────────────────────────────┤
│  [건물 필터: 전체 | 웨슬리홀 | 비전홀 | 다니엘홀]        │
│  [장소 필터: 드롭다운 멀티선택]                          │
│                                                     │
│  ┌─── 2026년 5월 ──────────────────────────────┐    │
│  │  < 이전  |  오늘  |  다음 >   [월|주|일]      │    │
│  ├───┬───┬───┬───┬───┬───┬───┤    │
│  │ 일 │ 월 │ 화 │ 수 │ 목 │ 금 │ 토 │    │
│  ├───┼───┼───┼───┼───┼───┼───┤    │
│  │   │   │   │   │ 1 │ 2 │ 3 │    │
│  │   │   │   │   │■본│   │   │    │
│  ├───┼───┼───┼───┼───┼───┼───┤    │
│  │ 4 │ 5 │ 6 │ 7 │ 8 │ 9 │10 │    │
│  │■소│   │■비│   │■본│   │   │    │
│  └───┴───┴───┴───┴───┴───┴───┘    │
│                                                     │
│  ■ 승인  ■ 대기중  ■ 거절                            │
│                                                     │
│  [ + 공간 신청하기 ]                                   │
└─────────────────────────────────────────────────────┘
```

## 2. 공간 신청 폼 (모달 또는 페이지)

```
┌─────────────────────────────────────────┐
│         공간 신청서                        │
├─────────────────────────────────────────┤
│                                         │
│  신청자 이름: [____________]              │
│  소속 부서/단체: [____________]            │
│                                         │
│  건물 선택: [드롭다운 ▼]                   │
│  장소 선택: [드롭다운 ▼] (건물 선택 시 필터)  │
│                                         │
│  날짜: [2026-05-05] (캘린더 피커)          │
│  시작 시간: [09:00 ▼]                     │
│  종료 시간: [10:00 ▼]                     │
│                                         │
│  반복 설정:                               │
│  ( ) 반복 없음                            │
│  ( ) 매주 반복 — [월 화 수 목 금 토 일]     │
│  ( ) 매월 반복 — 매월 [__]일               │
│  반복 종료일: [____________]               │
│                                         │
│  용도: [________________________]         │
│  연락처 (전화/이메일): [____________]       │
│  비고: [________________________]         │
│                                         │
│  [취소]              [신청하기]             │
└─────────────────────────────────────────┘
```

## 3. 일정 상세 보기 (캘린더 이벤트 클릭 시)

```
┌─────────────────────────────────────────┐
│         일정 상세                         │
├─────────────────────────────────────────┤
│                                         │
│  장소: 비전홀 1F - 믿음채플                 │
│  날짜: 2026년 5월 8일 (목)                 │
│  시간: 14:00 ~ 16:00                     │
│  신청자: 김철수                            │
│  소속: 청년부                             │
│  용도: 찬양팀 연습                         │
│  상태: [■ 승인됨]                         │
│                                         │

│                                         │
│  [닫기]                                  │
└─────────────────────────────────────────┘
```
